Technical details

The product uses two sets of Si7021 and CCS811 sensors interfaced with i2c to read the temperature, humidity and air quality values of a room's interior and exterior. It then processes the data to make a weighted decision using the relative values between the sets to create three output signals - window (open/closed), AC and heater. The processing depends on the selected mode and settings entered by the user using the app. The sensors and app communicate using an MQTT broker with encryption and authentication.

Our App

Always stay connected. Optimised for mobile.

In the App, you can choose between 5 different modes and decide which parameters to prioritise. You can choose 'default' but also 'energy saving mode' to save energy. In addition, you can chose which parameters have the highest priority for you. With snabb, you're in full control!

The settings screen allows the user to enter their desired temperature and humidity range, and select the operating mode. To avoid any negative health effects, humidity levels should stay between 30% and 50% over a longer period of time. If the humidity is too low, it can cause irritant cough.

The display panel shows current indoor temperature, humidity and air quality as well as the current status of the windows, AC and heaters. Temperature is measured in degrees Celsius, humidity as relative humidity expressed as a percentage and air quality as TVOCs in ppb. The user can enable automatic control.
